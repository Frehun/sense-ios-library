onlyFromDevice: (bool)  onlyFromDevice

Retrieve a number of values of a sensor from Common Sense. returns nrLastPoints of the latest values.

Parameters
nameThe name of the sensor to get data from
onlyFromDeviceWether or not to only look through sensors that are part of this device. Searches all sensors, including those of this device, if set to NO
nrLastPointsNumber of points to retrieve, this function always returns the latest values for the sensor.
Returns
an array of values, each value is a dictionary that describes the data point
